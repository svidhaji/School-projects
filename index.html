Git-komennot

Komennot, jotka tehdään alussa

  • git init
  • git remote add origin https://github.com/käyttäjä/repositoryn-nimi.git
  • Ensimmäisellä kerralla: git push -u origin master

Yleisimmät komennot

  • git init
  • git add .
  • git commit -m 'tekstiä'
  • git push

Muista tehdä README.md ja .gitignore
